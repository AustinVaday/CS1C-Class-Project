iRobotsTheBomb Project

CS1C Project, 2014.

Download this project as a .zip file Download this project as a tar.gz file

Source code above, video demo below. Enjoy!

Collaborators: Anthony Ramirez, Austin Vaday, Annie Raichev, Erik (Dan) Karlsson



Speacial Features:

  • User Features
    • Ability to request access to software (only particular people/organizations should be able to possess our bomb-destructing robots)
    • Ability to chat with a virtual Admin [incorporates Artificial Intelligence (AI)]
    • Ability to purchase multiple items via a shopping cart
    • Ability to view maintenance plans in case robot malfunctions
    • Ability to experience the software through "Guest" mode, while waiting for account access
    • Ability to write a review and view celebrity testimonials
  • Administrator Features
    • Ability to modify users as activated or deactivated through an easy-to-use display
    • Ability to modify (edit, add, or delete) user information through the "Master Mode"
    • Ability to search for a particular user
    • Ability to iterate through all users in a circularly linked list
  • General Features
    • Contingency handling (handling unexpected inputs)
